## 🚀 推荐方案：阿里云控制台远程连接（最简单）

### 详细操作步骤

#### 第一步：打开阿里云控制台

在浏览器中打开：https://ecs.console.aliyun.com/

#### 第二步：找到ECS实例

1. 在左侧菜单点击"云服务器ECS"
2. 找到实例名称（通常以 `iZ开头的）
3. 点击实例名称进入详情页

#### 第三步：远程连接

**在ECS详情页右侧**，找到"远程连接" → 选择以下任一方式：

**方式A：VNC连接**（推荐）
1. 点击"VNC连接"
2. 点击"立即连接"
3. 会打开一个新的浏览器窗口连接到服务器桌面

**方式B：Workbench连接**
1. 点击"Workbench"
2. 点击"打开终端"
3. 会打开一个Web终端

**方式C：会话管理**
1. 点击"会话管理"
2. 点击"打开会话"

---

### 第四步：连接成功后执行命令

**在远程终端中复制粘贴以下全部命令**：

```bash
cd /opt/shared_cfc && source venv/bin/activate && python crawler.py
```

**注意**：复制时确保复制了完整的三条命令。
